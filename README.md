# pqcscan v2

Post-Quantum Cryptography (PQC) readiness scanner.

Fresh design, single Python binary with bundled web UI + headless CLI, runs locally on Linux / Windows / macOS servers. Fully FOSS, offline-first.

**Status:** design phase — implementation has not started.
