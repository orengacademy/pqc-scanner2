How to read this page.

aegis-style traffic-light readiness

Each finding is bucketed into a traffic-light band based on the algorithm in use, the probe family, and whether the host software has a PQC upgrade path.

  • Green PQC or hybrid PQC in use (ML-KEM, ML-DSA, SLH-DSA, X25519MLKEM768).
  • Yellow Classical crypto today, but the software / vendor supports a PQC upgrade.
  • Red Classical crypto with no PQC roadmap. Highest risk.
  • Grey Not yet scanned, unreachable, or purely informational.

Hover any algorithm in the findings list for a plain-English explanation. Mappings: NIST IR 8547, NIST FIPS 203/204/205, MyKripto, BUKUKERJA, NACSA.
